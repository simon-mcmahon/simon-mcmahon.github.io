# Simon McMahon - Blog and Resume

This repository holds the source for [simonmcmahon.com](https://simonmcmahon.com), my personal site for sharing blog posts, career updates, and a living resume built with the Minimal Mistakes Jekyll theme.

## What you will find
- **Blog posts** covering technical learning, self hosting experiments, and coding projects stored in `_posts/`
- **About page** that introduces my background and day to day security focused IT support work
- **Resume page** published at `/resume` with updates in progress
- **Tag, category, and year archives** for browsing past posts
- **Navigation and data files** in `_pages/` and `_data/` that drive menus, layouts, and site metadata

## Local development
1. Install Ruby and Bundler (Ruby 3.x recommended).
2. Install dependencies:
   ```bash
   bundle install
   ```
3. Run the site locally:
   ```bash
   bundle exec jekyll serve
   ```
4. Visit `http://localhost:4000` to preview changes.

## Deployment
The site is configured for GitHub Pages. Updates merged to the default branch are built and served automatically using the settings in `_config.yml` and `CNAME`.
